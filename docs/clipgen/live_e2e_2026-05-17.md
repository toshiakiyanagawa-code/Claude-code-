## To Run Real Live E2E

Create `.env` from `.env.example`, then set:

```text

```

For quota control, do not run the full default seed set blindly. The current seed list expands to many YouTube search requests. Prefer a narrow live smoke script or a temporary one-query seed file before enabling daily live runs.
