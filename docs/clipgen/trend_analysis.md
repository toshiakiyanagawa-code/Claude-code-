## 1. 結論サマリー（量産戦略）

**「人気切り抜きチャンネル」上位の共通点を抽出した、量産すべき動画フォーマット:**

1. **形式は基本ショート（60 秒以下）**。再生上位はほぼ全てショート。([factcheckcenter.jp](https://www.factcheckcenter.jp/explainer/politics/election-2026-youtube/))
2. **狙う対象人物は「再生数の磁場」がある政治家**。現状の磁場 No.1 は **高市早苗**（個人名関連で 4.46 億回視聴）。([go2senkyo.com](https://go2senkyo.com/articles/2026/02/02/128635.html))
3. **題材は「政策」ではなく「印象」「快/不快」**。記者会見・国会・街頭での"瞬間芸"を切り出す。([factcheckcenter.jp](https://www.factcheckcenter.jp/explainer/politics/election-2026-youtube/))
4. **タイトルは煽り型 + 【絵文字括弧】 + 短文**。例:
    - `【総理の朝】お化粧パタパタってやってます！`
    - `【爆笑】自民党候補の決め台詞に爆笑してしまう大臣`
    - `【※サナエ人気が炸裂】高市人気！`
    - `【絶句】安住淳氏、親族の訴えを鼻で笑う`
    - `【よねやんパニック】中道改革連合・米山隆一、路肩の雪を道路にぶちまけ`
5. **素材ソースは TV を絶対に避け、以下のホワイトリストに限定**:
    - 政党公式 YouTube（記者会見、街頭演説、ライブ配信）
    - 国会 TV / 衆議院・参議院インターネット中継（政府著作物・引用可能）
    - 地方議会の公式議会映像（公開済み・引用可能、例: 安芸高田市の石丸切り抜きで実証 — 1188 万再生）
    - 議員本人の公式 YouTube チャンネル（**切り抜き OK 明記** のみ）
    - 政治系 YouTuber（**チャンネル概要欄で切り抜き許諾** が明記されているもの）
6. **第三者投稿が市場の 85%（参院選では 92.5%）**を占有、うち切り抜き系チャンネルが 73.3%。実質「個人 / 匿名」のレッドオーシャンだが、アルゴリズム上**登録者数より動画単位の質**で勝負できる。([factcheckcenter.jp](https://www.factcheckcenter.jp/explainer/politics/election-2026-youtube/), [go2senkyo.com 鈴木邦和 解説動画](https://www.youtube.com/watch?v=HYUe6mRpwVs))

## 3. 「伸びる動画」共通法則（抽出版）

選挙ドットコム副編集長・鈴木邦和氏の YouTube ポジネガ解析（[ABEMA TIMES 解説](https://news.yahoo.co.jp/articles/f12cb085e7cefdb4dfd5c9e34e855e09a9cf848f) ＋ [鈴木邦和解説動画](https://www.youtube.com/watch?v=HYUe6mRpwVs) ＋ [選挙ドットコム go2senkyo 記事](https://go2senkyo.com/articles/2026/02/02/128635.html)) ほかから整理。

### 3.1 イデオロギー的傾向
- **保守的コンテンツが伸びやすい**（ポジネガ解析）
- 自民党関連は **ポジティブ判定 6 割超**
- 中道改革連合の上位 20 本は **20 本全てネガティブ判定**
- 「高市 OK だが自民党は NG」「高市を勝たせるには自民党を過半数割れさせろ」型のロジックがバズる

### 3.2 動画フォーマット（テンプレ化可能）

| パターン | 構成 | 例 |
|---|---|---|
| **快フォーマット** | 政治家の人間味・笑顔・親しみ | 「総理の朝、お化粧パタパタ」「決め台詞に大臣爆笑」 |
| **絶句フォーマット** | 失言・態度・暴言の瞬間 | 「親族の訴えを鼻で笑う」「失敗した瞬間」 |
| **パニックフォーマット** | 物理的失敗、しどろもどろ | 「雪を道路にぶちまけ」 |
| **論破フォーマット** | 他者を論理的に追い詰める瞬間 | 石丸 × 議会、ひろゆき切り抜き |
| **対立構造解説フォーマット** | 「○○党 vs ○○党」の図解＋字幕 | 「あの政党が苦戦の理由」 |

### 3.3 編集・サムネ・タイトル
- **接続語（あの／その／えーと）は全カット**で 1.5〜2 倍速感を出す（[東京新聞 385845](https://www.tokyo-np.co.jp/article/385845)）
- **テロップは座布団入り、書式統一**（[note クーニャ記事](https://note.com/happy_roses398/n/na36765beab98) で紹介の Vrew パターン）
- **サムネは「強気の書き方」** だが、嘘・釣り見出しは NG（信用毀損で BAN 連鎖のリスク）
- **タイトル先頭に【◯◯】を 1〜2 個**入れて目を引く
- **動画長は短く**: ショートは 60 秒以内、通常動画でも 3〜8 分が多い
- **「100 万再生超だがコメント激少」は広告ブースト動画なのでパクっても伸びない**（[factcheckcenter.jp](https://www.factcheckcenter.jp/explainer/politics/election-2026-youtube/)）

### 3.4 アルゴリズム上の有利点
- 新規チャンネルでも登録者数と無関係に伸びる
- 「同じテーマを多視点で連投」が効く（次の動画にレコメンドされやすい）

## 4. 著作権・素材ソース安全ライン

### 4.1 ホワイトリスト（基本 OK と判断する素材ソース）

| カテゴリ | 例 | 根拠 |
|---|---|---|
| 政党公式 YouTube | 自民党, 立憲, 維新, 国民, 参政党, れいわ, 中道改革連合, 公明, 共産 ほか公式アカウント | 公開記者会見・街頭演説は公的言論。引用範囲を守れば一般に許容（[tokyo-np 385845](https://www.tokyo-np.co.jp/article/385845)） |
| 国会中継 | 衆議院/参議院インターネット審議中継、国会 TV | 政府著作物・引用容易 |
| 地方議会公式映像 | 各市議会の YouTube / 公式配信 | 公開映像、石丸切り抜きで前例化 |
| 政治家本人の公式 YouTube | 高市早苗, 玉木雄一郎, 石丸伸二, 神谷宗幣 ほか | チャンネル概要欄に「切り抜き OK」明記の場合のみ |
| 切り抜き OK 明記の政治系 YouTuber | 例: ReHacQ, 一月万冊 など (概要欄で許諾を都度確認) | 各チャンネルポリシー |

### 4.2 ブラックリスト（絶対不使用）

- **TV 局放送番組の録画・転載**: 民放/NHK の通常番組、ニュース番組、ワイドショー
  - ただし TV 局公式 YouTube に上がっているフリー公開クリップ（ANN、TBS NEWS DIG、テレ朝など）は**チャンネル設定で「埋め込み許可」されていても切り抜き再アップは不可**。引用扱いの短尺コメント動画は別途検討。
- **新聞社の有料記事画面、配信動画**
- 切り抜きを明示的に禁止しているチャンネル（後述 NG リスト）
- 一般人を特定可能な形で晒す素材
- 公人でも私生活シーンが映る素材

### 4.3 チャンネル別 切り抜き許諾の確認手順
1. チャンネル「概要」欄の規約を読む（「切り抜き禁止」「無断転載禁止」の文言確認）
2. 動画の説明欄を読む（個別動画単位で許諾/不許諾の場合あり）
3. 直近半年内に削除申立てや CCM (Content ID Match) の事例がないか YouTube 検索
4. グレーなら**使わない**（チャンネル BAN リスクを最優先で回避）

### 4.4 NG チャンネル初期リスト（要更新）
以下は「切り抜き禁止」または「権利者対応が厳しい」として一般的に知られているチャンネル。`src/clipgen/data/blocklist.json` で管理し、随時追記する。

- TV 局公式チャンネル全般（TBS NEWS DIG, ANN, FNN, 日テレ, テレ東 BIZ, NHK, MBS, カンテレ, etc.）
- 新聞社公式チャンネル（朝日, 読売, 毎日, 産経, 日経, 共同通信 など）
- 個別議員 / インフルエンサーで切り抜き NG を明記しているもの（運用時に随時追加）

## 6. データ取得戦略（実装ガイド）

### 6.1 一次データ
- **YouTube Data API v3** (`search.list`, `videos.list`, `channels.list`)
  - 政治系シードクエリで日次検索
  - 公開 1〜14 日以内 & 再生数 > 5 万 を一次フィルタ
- **チャンネル単位ホワイトリスト探索**: 各政党公式・議員公式・国会 TV のチャンネル ID から `playlistItems.list` でフィード取得

### 6.2 スコアリング
- `views_per_hour` = 再生数 / 公開後の経過時間 → 「初速」を最重視
- `like_ratio` = いいね / 再生数 → 0.02 以上を健全動画ライン
- `comment_density` = コメント / 再生数 → 1e-4 未満は広告ブースト動画疑い→除外
- `freshness_boost` = exp(-経過時間/48h) → 直近 2 日に強い重み
- `keyword_boost` = タイトルに 5.1 のホット人物名 / フォーマット語が含まれたら +α

### 6.3 出力
- `output/candidates_YYYYMMDD.json`
- 1 候補あたり: 元動画 URL / チャンネル / タイトル / 推奨切り抜きハイライト窓 / 提案タイトル案 / リスクフラグ

## 7. リスクと運用ルール

| リスク | 軽減策 |
|---|---|
| 名誉毀損 / 信用毀損 | タイトルは煽り OK だが**事実に反する記述は NG**。「鼻で笑う」など映像内の事実描写は OK |
| プライバシー侵害 | 公人の公的発言のみを対象、私生活映像は除外 |
| Content ID マッチ → 収益化停止 | TV 局・新聞・音楽の素材ゼロを徹底 |
| チャンネル BAN | NG チャンネルリスト ([src/clipgen/data/blocklist.json](../../src/clipgen/data/blocklist.json)) と切り抜き許諾チェッカーを必ず通す |
| フェイクニュース・誤情報 | 字幕で発言を改変しない、文脈をねじ曲げる切り出しを避ける |
| 選挙期間中の規制論 | 公職選挙法・将来の規制動向を継続ウォッチ ([tokai-tv.com](https://www.tokai-tv.com/tokainews/feature/article_20250625_41022)) |
